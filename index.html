Functions

  • Deauth: Disconnect all devices from any Wi-Fi network, with the ability to target specific devices for disconnection.
  • Beacon: Flood networks with fake access points to confuse users by creating multiple false Wi-Fi names, making it harder to locate the genuine network.
  • Probe: Inject a large number of users to crash Wi-Fi strength, causing connected devices to appear connected but with no internet access.
  • Packet Monitor: Capture and analyze network traffic.
  • Clock: Display time
  • Scanning Wi-Fi APs and Stations: Identify available Wi-Fi access points and connected devices.
  • Cloning SSIDs: Clone Wi-Fi network names to flood networks with fake SSIDs, enhancing Beacon and Probe attacks.
  • Random Mode: Generate random fake SSIDs to flood networks with Beacon and Probe attacks for testing network resilience.

Wi-Fi Device Information

The G-Shark Deauther Watch can display detailed information about detected Wi-Fi devices:

  • Wi-Fi Name: Network name (SSID).
  • Encryption Type: Security protocol used (WEP, WPA, WPA2, etc.).
  • RSSI: Signal strength indicator.
  • Channel Number: Wi-Fi channel used by the network.
  • SSID: Service Set Identifier (network name).
  • Connected Devices: List of devices currently connected to the network.
